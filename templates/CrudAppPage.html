ANÁLISIS Y GESTIÓN DIGITAL DE RUTAS Y ESTACIONES DEL SISTEMA TRANSMILENIO

MEDIANTE UNA API DESARROLLADA CON FASTAPI Y POSTGRESQL

Por: Juan Esteban Tovar Vargas 67001350

Objetivo

Desarrollar una API con FastAPI que permita registrar, consultar, filtrar y administrar rutas y estaciones del sistema de transporte Transmilenio, almacenando los datos en una base de datos PostgreSQL y preparándola para despliegue en la nube (Render).

Problema / situación / oportunidad:

El sistema Transmilenio es fundamental para la movilidad en Bogotá, pero el acceso a datos organizados, actualizables y reutilizables por terceros (desarrolladores, analistas, ciudadanos) es limitado. La mayoría de los datos están dispersos o requieren limpieza antes de ser usados. Esto representa una oportunidad para desarrollar una solución moderna y flexible que permita administrar y consultar estos datos mediante una API, facilitando el desarrollo de futuros sistemas o análisis que contribuyan a mejorar la eficiencia del transporte público.
